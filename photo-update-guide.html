How to Update Gallery Photos

A step-by-step guide for swapping photos and adding new grid boxes — no code required.

Part 1 — Upload a Photo to GitHub

Every photo on the site lives in the images/ folder inside the GitHub repository. Uploading a new photo there makes it available to any page.

Steps to Upload

1
Go to github.com/modernluxeflowers2026/modernluxeflowers.com and sign in.
2
Click the images folder.
3
Click Add file → Upload files (top-right of the file list).
4
Drag your photo in, or click to choose it. Name it before uploading — the filename is what goes in the page code.
5
Scroll down and click Commit changes. The photo is now live at https://modernluxeflowers.com/images/your-filename.jpg.

Photo tips: Export at JPG quality 85. Portrait images should be at least 800×1100 px. Landscape images at least 1400×700 px. Keep file names lowercase with hyphens, no spaces — e.g. spring-bridal-blush.jpg.

Part 2 — Swap an Existing Photo

The easiest update: replace an old photo with a new one while keeping the grid layout exactly the same.

Option A — Same filename (zero code changes)

Upload your new photo with exactly the same filename as the one it replaces. GitHub will overwrite the old file. Refresh the page and the new photo appears automatically — no HTML editing needed.

Option B — New filename (edit one line of HTML)

1
Upload the new photo to images/ as described in Part 1.
2
In the GitHub repo, navigate to the collection HTML file you want to edit (e.g. luxe-love-unique-flower-arrangements.html) and click the pencil ✎ Edit icon.
3
Press Ctrl+F (or Cmd+F on Mac) and search for the old filename — for example old-photo.jpg.
4
Replace the old filename with the new one. Only change the filename part, leave the rest of the src URL intact.
5
Click Commit changes. The site updates in about 30 seconds.

What a src line looks like:
src="https://modernluxeflowers.com/images/spring-bridal-blush.jpg"
Only the filename at the end changes. Everything before images/ stays the same.

Part 3 — Add a New Grid Box

The three collections use two different grid styles. Find the one that matches and follow those steps.

Which grid style is on each page?

PageGrid Style
Boxful ArtUniform Grid — equal 3-column squares
Luxe LoveMosaic — hero left, bento right, portrait row below
Petals & StemsMosaic — hero left, 2×2 bento right, portrait row below

Adding a box to the Uniform Grid (Boxful Art)

The Boxful Art page has a simple repeating grid. Each photo is a gallery-item block. New boxes are added to the end and the grid fills automatically.

1
Upload your new photo to images/ (Part 1).
2
Open gift-giving-boxful-art.html in the GitHub editor.
3
Search for </div><!-- /gallery-grid -->. Just above that closing tag, paste this block (replacing the filename):
<div class="gallery-item">
  <img src="https://modernluxeflowers.com/images/YOUR-PHOTO.jpg" alt="Description of photo" loading="lazy">
</div>
4
Click Commit changes. The new box appears at the end of the grid.

The 3-column grid fills left to right, top to bottom. You can add as many boxes as you like — the grid expands automatically.

Adding a box to the Mosaic layout (Luxe Love / Petals & Stems)

The Mosaic pages have two sections: a fixed top section (hero + bento), and a bottom portrait row. The bottom row is where you add new photos.

1
Upload your new photo to images/ (Part 1).
2
Open the collection HTML file in the GitHub editor.
3
Search for </div><!-- /mosaic-bottom -->. Just above that closing tag, paste this block:
<div class="cell">
  <img src="https://modernluxeflowers.com/images/YOUR-PHOTO.jpg" alt="Description of photo" loading="lazy">
</div>
4
Click Commit changes. The new portrait cell appears in the bottom row.

The bottom row always has 3 equal portrait columns. If you add a 4th photo it wraps to a second row — also 3 columns. The carousel lightbox automatically picks up all new photos.

Part 4 — Change a Hero or Bento Photo

The top section of Luxe Love and Petals & Stems has a tall hero photo on the left and a grid of smaller photos on the right. These are fixed-position cells — you can swap the photo inside each one but the layout stays the same.

Finding the right cell in the code

Each cell has an HTML comment above it showing which position it is. Open the file in GitHub, press Ctrl+F, and search for the current filename you want to replace. Change only that filename, then commit.

The hero photo is at the very top of the mosaic section and is usually the tallest image in the layout — at least 800×1400 px works best here. Bento cells are square-ish; any portrait crop at 800×900 px works well.

Part 5 — Quick Reference

HTML block for Uniform Grid (Boxful Art)

<div class="gallery-item">
  <img src="https://modernluxeflowers.com/images/FILENAME.jpg" alt="Short description" loading="lazy">
</div>

HTML block for Mosaic bottom row (Luxe Love / Petals & Stems)

<div class="cell">
  <img src="https://modernluxeflowers.com/images/FILENAME.jpg" alt="Short description" loading="lazy">
</div>
