Posting Cadence

Sequence logic: Lead with value, earn trust, then sell. Never open a new account with a product pitch.
Week Day Script CTA Type Rationale
Week 1 Tue Script 1 — Keepa filter Affiliate Pure value first. Teaches something real.
Week 1 Fri Script 4 — Keepa + SellerAmp Affiliate Builds on Script 1 — deepens tool credibility.
Week 2 Tue Script 5 — SellerAmp standalone Affiliate Reinforces tool before product ask.
Week 2 Fri Script 2 — AI Playbook Gumroad $17 Trust is established. Now sell.
Week 3 Tue Script 3 — Aura repricer Affiliate Reset to value after sell.
Week 3 Fri Repeat or new Script 2 variant Gumroad $17 Second product exposure to growing audience.
  • Never post two sell scripts back-to-back.
  • Minimum 1 value post between every Gumroad CTA.
  • Post Tue and Fri only until you hit 500 followers — consistency beats volume at this stage.
  • After 500 followers, move to 3x/week (Mon, Wed, Fri).
TikTok launch (June 16): Start with Script 1 on launch day — same value-first logic. Don't mirror Instagram post-for-post; stagger by 2–3 days so each platform gets a "fresh" post even if the content is identical.
